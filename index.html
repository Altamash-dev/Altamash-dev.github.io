Object Oriented Programming Project

Java Based Mini Project Website

About The Project

This website is created for demonstrating the concepts of Object Oriented Programming using Java. The project explains important OOP principles such as Classes, Objects, Inheritance, Polymorphism, Abstraction and Encapsulation.

OOP Concepts Used

Concept Description
Class Blueprint for creating objects
Object Instance of a class
Inheritance Acquiring properties from another class
Polymorphism One method with multiple forms
Encapsulation Binding data and methods together
Abstraction Showing only important details
